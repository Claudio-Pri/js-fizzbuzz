#

-Conto fino a 100

Se, il numero è multiplo di tre e di cinque stampo "FizzBuzz"

Altrimenti Se, il numero è multiplo di tre stampo "Fizz"

Altrimenti Se, il numero è multiplo di cinque stampo "Buzz"
